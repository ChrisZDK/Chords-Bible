# 🎹 Chordyssey

**Chordyssey** is an interactive music-learning app designed to help users explore chords with fun in a arcade-inspired environment.

Users can build chords, select root notes, explore chord types, view formulas, see chord notes, and interact with a visual piano interface.

🌐 Live site: http://chordyssey.com/ 

---

## ✨ Concept

Chordyssey is built as a musical adventure map. The goal is simple: make chord learning clearer and magical.
